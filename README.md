# Система для получения космических данных

Для запуска проекта на локальном устройстве необходимо

1) Склонировать репозиторий:
``git clone https://github.com/YatagarasuGames/Frameworks3.git``

2) Запустить контейнеры бд и redis:
``docker-compose up -d``
3) Запустить фронт фронтэнд: Находясь в корневой папке проекта:
``cd Frameworks3Frontend``
``dotnet run``

4) Запустить бэкэнд
``cd Frameworks3``
``dotnet run``

Сайт будет доступен на localhost:5001

## Можно просматривать космические события на выбранной локации(вписав широту и долготу), перейдя на вкладку "Астрономические события"

## Можно просматривать местоположение МКС в реальном времени на вкладке "Данные МКС"

## Можно просматривать данные OSDR NASA на вкладке "OSDR"
